# myWebStore
test task for Baits

[Demo](https://fluxet.github.io/myWebStore/)

## Install

```sh
$ npm install

(or $ npm install --legacy-peer-deps)
```

## Build

```sh
$ npm run build
# open index.html in 'dist' directory
```

## Open on loacalhost

```sh
$ npm run start
# open http://localhost:8080
```
